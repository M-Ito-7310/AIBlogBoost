# AIBlogBoost

AI を活用した高品質なブログ記事作成支援ツール。直感的な6ステップウィザードで魅力的なコンテンツを生成します。

![Vue.js](https://img.shields.io/badge/vue.js-3.5-green)
![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue)

**言語**: [English](README.md) | 日本語

## 概要

AIBlogBoost は、Google Gemini AI を活用してブログ記事作成を支援するフロントエンド専用 SPA です。構造化されたワークフローを通じて魅力的なブログ記事を作成できます。モダンな Web 開発技術と AI 統合を実証するポートフォリオプロジェクトとして開発されました。

## 特徴

### 🎯 **6ステップ記事作成ワークフロー**
1. **ジャンル選択** - ビジネス、テクノロジー、ライフスタイルなど豊富なジャンル
2. **テーマ設定** - 記事のトピックとターゲット読者を指定
3. **AI アイデア生成** - Gemini AI による5つの創造的なアイデア提案
4. **多様な草案作成** - 異なるトーン（プロフェッショナル、カジュアル、教育的）で3つの草案を生成
5. **インテリジェント組み合わせ** - 各草案の優れた部分を統合して完成度の高い記事を作成
6. **エクスポート＆保存** - 複数の形式でダウンロード、ローカル履歴に保存

### ⚡ **AI 駆動機能**
- **Google Gemini 1.5 Flash** 統合による高品質なコンテンツ生成
- ジャンルとテーマに基づくコンテキスト対応記事提案
- 多様なコンテンツスタイルに対応した複数の文体オプション
- ユーザー指定の指示による知的なコンテンツ組み合わせ

### 📱 **モダンなユーザー体験**
- **レスポンシブデザイン** - デスクトップ、タブレット、モバイルでシームレスに動作
- **ダーク/ライトテーマ** - システム設定検出による自動テーマ切り替え
- **プログレッシブ UI** - 進捗追跡付きのステップバイステップウィザード
- **ローカルデータ永続化** - ブラウザストレージに記事と設定を保存

### 📄 **エクスポート＆管理**
- **複数エクスポート形式**: Markdown、HTML、プレーンテキスト
- **記事履歴** - 過去に作成した記事の閲覧と管理
- **お気に入りシステム** - 重要な記事を素早くアクセス
- **クリップボードコピー** - 簡単共有機能

## 技術スタック

### フロントエンドアーキテクチャ
- **Vue.js 3** Composition API と TypeScript
- **Tailwind CSS** ユーティリティファーストスタイリングとレスポンシブデザイン
- **Pinia** リアクティブ状態管理

### AI 統合
- **Google Gemini API** (1.5 Flash モデル) によるコンテンツ生成
- **直接 API 統合** - ユーザー提供の API キーを使用するフロントエンド専用アーキテクチャ
- **インテリジェントプロンプティング** - 日本語コンテンツ作成に最適化されたプロンプト

### データ管理
- **LocalStorage** 記事履歴とユーザー設定
- **クライアントサイド処理** - サーバー不要、完全なプライバシー保護
- **エクスポートサービス** - ブラウザベースのファイル生成

## クイックスタート

### 前提条件
- Node.js 18+ (推奨: 20+)
- Google Gemini API キー ([API キー取得](https://makersuite.google.com/app/apikey))
- モダンなウェブブラウザ

### インストール

1. **リポジトリをクローン**
   ```bash
   git clone https://github.com/M-Ito-7310/AIBlogBoost.git
   cd AIBlogBoost
   ```

2. **依存関係をインストール**
   ```bash
   cd frontend
   npm install
   ```

3. **開発サーバーを起動**
   ```bash
   npm run dev
   ```

4. **ブラウザで開く**
   http://localhost:5173 にアクセス

5. **API キーを設定**
   - 設定ページに移動
   - Google Gemini API キーを入力
   - 記事作成を開始！

## 使用ガイド

### はじめに
1. **API 設定**: 設定で Google Gemini API キーを設定
2. **作成開始**: ホームページの「記事作成を開始」をクリック
3. **ウィザードに従う**: 6ステッププロセスの各段階を完了
4. **エクスポート＆保存**: 完成した記事をお好みの形式でダウンロード

### API キー管理
- API キーはブラウザにローカル保存されます
- セキュリティのためキーは暗号化されて保存されます
- 設定でいつでも API キーの変更や削除が可能です

### 記事管理
- すべての記事はブラウザにローカル保存されます
- 素早いアクセスのために記事をお気に入りに登録
- 複数の形式で記事をエクスポート
- 記事履歴を検索

## アーキテクチャの決定

### フロントエンド専用設計
- **プライバシー**: ユーザーデータが外部サーバーに送信されない
- **シンプルさ**: バックエンドの複雑性がない単一コードベース
- **コスト効率**: サーバーインフラが不要
- **スケーラビリティ**: グローバルパフォーマンス向上のための CDN ベース配信

### Google Gemini 統合
- **品質**: 高品質な日本語コンテンツ生成
- **コスト効率**: 1日1,500リクエストの無料枠
- **信頼性**: 高可用性の本番対応 API
- **柔軟性**: 最大制御のための直接 API 統合

### 状態管理戦略
- **Pinia** TypeScript サポート付きリアクティブ状態管理
- **LocalStorage** セッション間でのデータ永続化
- **モジュラーストア** 関心の分離によるクリーンな設計

## セキュリティ＆プライバシー

### データハンドリング
- すべてのデータはブラウザ内でローカル処理
- API キーは暗号化してローカルストレージに保存
- 第三者にユーザーデータを送信しない
- コンテンツと設定の完全な制御

### API キーセキュリティ
- ブラウザ暗号化でキーを保存
- 許可されていないエンドポイントに送信されない
- ユーザーはいつでもキーを削除可能
- API キー状態の明確な表示
